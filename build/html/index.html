Endpoints¶

User endpoints¶

POST /marketplace/api/sign-up¶

Sign up a new user to the database.

Example -

Request JSON Object:
  
{
    "username": "johndoe",
    "password": "gaslbj3l4i",
    "email": "johndoe@email.com"
}
Response JSON Object:
  
{
    "message": "User signed up successfully",
    "new_user":
        {
            "username": "johndoe",
            "email": "johndoe@email.com"
            "cart": [
                "63f6-bj6m-345k",
                "354g-3427-nb38"
            ]
        }
}
Status Codes:
  • 201 Created - New user signed up
  • 400 Bad Request - Malformed request
POST /marketplace/api/sign-in¶

Perform sign in of a user.

Example -

Request JSON Object:
  
{
    "username": "johndoe",
    "password": "gaslbj3l4i"
}
Response JSON Object:
  
{
    "message": "Login successful"
}
Status Codes:
  • 200 OK - Login successful
  • 404 Not found - Username not found
  • 400 Bad request - Incorrect password

POST /marketplace/api/complete-cart¶

Complete the given user’s cart. The cart is used to generate an order. The user’s cart is cleared. The inventories of the products purchased by the user are decremented.

Return the order information along with the product(s) whose inventories were decremented.

Example -

Request JSON Object:
  
{
    "username": "Midoriya"
}
Response JSON Object:
  
{
    "affected_products": [
        {
            "inventory_count": 7,
            "price": 7.99,
            "title": "Orange cupcake",
            "uri": "http://localhost:5000/marketplace/api/product/f4ad5da8-2cc5-4ec0-86f3-4c02367c082f"
        }
    ],
    "order": {
        "amount": 15.98,
        "order_id": "baef750f-9c54-41f8-a7a5-647640ff62d1",
        "products": [
            {
                "inventory_count": 7,
                "price": 7.99,
                "title": "Orange cupcake",
                "uri": "http://localhost:5000/marketplace/api/product/f4ad5da8-2cc5-4ec0-86f3-4c02367c082f"
            },
            {
                "inventory_count": 7,
                "price": 7.99,
                "title": "Orange cupcake",
                "uri": "http://localhost:5000/marketplace/api/product/f4ad5da8-2cc5-4ec0-86f3-4c02367c082f"
            },
        ],
        "username": "Uraraka"
    }
}
Status Codes:
  • 200 OK - Cart completed
